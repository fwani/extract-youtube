# Extract YouTube

Extract Video & Audio file from YouTube

# Description

- choose 1080p video
- choose maximum audio ABR(Average Bit rate)

# Usage

```bash
python main.py -u <youtube url>
```

# Output

- remove special characters of youtube title
  - `,`, `|`, `#`, `'`, `?`

```base
<youtube title>.mp3
<youtube title>.mp4
```
